# File and Folder Naming Policy

## Naming Conventions
- **Use lowercase**: All file and folder names must be in lowercase
- **Use underscores**: Use underscores (`_`) to separate words, not hyphens (`-`)

## Examples
- **Correct**: `module_name`
- **Wrong**: `module-name`, `ModuleName`, `module_name_2`

## When Creating New Files/Folders
- All new files and folders must follow this naming convention
- Python modules and packages should use underscores (Python convention)
- Shell scripts should use underscores for consistency
